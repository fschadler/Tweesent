Frequently Asked Questions

What is a Sentiment Analysis?

Sentiment Analysis is a technique used in text mining. It may, therefore, be described as as text mining technique for analyzing the underlying sentiment of a text message, i.e., a Tweet. Twitter sentiment or opinion expressed through it may be positive, negative or neutral.

How do you calculate the Sentiment Score?

We use a Python Module called VaderSentiment for the calculation of the Sentiment Score. Before analyzing each Tweet, our programs strips all irrelevant data (Links, special characters, ...) from the text, which could affect the evaluation process. After rating the Tweet, our system performs additional calculations, which can be seen on our Output Page.

How can I use this website?

It is fairly simple to use. All you have to do is head over to our Homepage and input the Searchterm or Hashtag you would like us to analyse for you. Additionally you can define how many Tweets should be considered.

Where does the data come from?

We use the Twitter API to directly access the latest Tweets from Twitter based on your input.
